# Quotation Template Guide

This guide provides templates and examples for creating professional service quotations.

## 📋 Quotation Information Checklist

### Essential Information to Include

#### 1. **Company Information**
- Company name and logo
- Business address
- Phone number
- Email address
- Website
- Tax ID / Business registration (if applicable)

#### 2. **Client Information**
- Client name
- Client company (if applicable)
- Contact email
- Contact phone
- Delivery/billing address

#### 3. **Quotation Details**
- Unique quotation number
- Date of issue
- Validity period (e.g., "Valid for 30 days")
- Project type
- Service type

#### 4. **Services Breakdown**
For each service item:
- Service name
- Detailed description
- Quantity
- Unit price
- Line total

#### 5. **Pricing Summary**
- Subtotal
- Discounts (if any)
- Tax rate and amount
- **Grand total**

#### 6. **Terms & Conditions**
- Payment terms (e.g., "50% deposit, 50% on completion")
- Delivery timeline
- Revision policy
- Cancellation policy
- Warranty information

## 📧 How to Send Quotations

### Via Email
1. Export quotation as PDF
2. Write a professional email
3. Attach the PDF quotation
4. Include a brief summary in the email body
5. Request confirmation of receipt

### Via Telegram (using your existing system)
1. Generate quotation PDF
2. Send as document with caption
3. Include key highlights in message
4. Request acknowledgment

### Best Practices
- Always send as PDF (never editable format)
- Include a clear call-to-action
- Set a follow-up reminder
- Track quotation status (Sent, Viewed, Accepted, Declined)

## 🔧 Integration with QuoteForm

To auto-generate quotations from quote form submissions:

1. **Capture form data** from `QuoteForm.tsx`
2. **Pre-fill** `QuotationBuilder` with client information
3. **Add services** based on selected service type
4. **Generate quotation** and send to client
5. **Track status** in your CRM or project management system

## 🎨 Customization Tips

1. **Add your branding**: Logo, colors, fonts
2. **Include portfolio samples**: Attach relevant work samples
3. **Personalize**: Reference specific project details
4. **Clear next steps**: Tell client how to accept
5. **Multiple options**: Offer tiered pricing (Basic, Standard, Premium)

---

## ⚠️ Important Notes

- Always specify what's included AND what's not
- Define revision limits clearly
- State payment terms upfront
- Include validity period (typically 30 days)
- Keep a record of all quotations sent
- Follow up within 3-5 business days if no response
